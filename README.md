# Airflow-ETL
Running ETL workflow in Apache Airflow

## Docker commands

### Initialize airflow instance
```bash
docker-compose up airflow-init
```
```bash
docker-compose up 
```
### check if containers are up and running
```bash
docker ps
```
### shut down services
```bash
docker-compose down
```
### execute bash commands in container
```bash
docker exec <container ID>
```
